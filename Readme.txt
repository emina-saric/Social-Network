Konfiguracija projekta na Localhost-u :

1. Pokrenuti projekat, staviti SocialNetwork.Web kao startup ako vec nije
2. Build projekat i Update-Database
3. Pokrenuti Aplikaciju

NOTE : Ako ConfirmEmail ne radi, potrebno je bazu izbrisati iz ObjectExplorer i Server Explorer, pa ponovo uraditi Update-Database.


Prilikom prvog pokretanja aplikacije, mogucnosti su :

Logiranje sa sljedecim korisnickim akauntima :

Username : SuperAdminUser
Password : SuperAdminUser123


Username : AdminUser
Password : AdminUser123


Dalje mogucnosti :
1. Izmijena jezika : U gornjem Desnom uglu se pritisne "BA", promijena u "EN"
2. Prijava Korisnika
3. Registracija novog Korisnika
4. Resetovanje izgubljene sifre
5. Dodavanje,Brisanje i Editovanje obavijesti
6. Pretraga korisnika i dodavanje istih u prijatelja
7. Prikaz svih obavijesti od prijatelja
8. Izmijena vlastitih podataka
9. Izmijena vlastite sifre
10. Izmijena profilne slike
11. Pretraga i brisanje prijatelja
12. Brisanje profila
13. Odjava korisnika

Admin specificne mogucnosti :
1. Brisanje svih korisnika osim onih sa rolom SuperAdmin
2. Izmijena svih korisnika osim onih sa rolom SuperAdmin
3. Dodavanje novih korisnika
4. Prikaz nekih statistickih podataka (charts)
